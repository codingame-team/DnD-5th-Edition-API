

## Game Locations

### The Tavern
A gathering place where players can:
- Recruit new characters
- Manage character roster
- Add characters to party
- Inspect character details
- Remove characters from party

Features:
- Character roster management
- Party size limit (6 members)
- Character inspection system
- Dynamic table updates
- Selection tracking

### The Inn
A resting place where characters can:
- Heal injuries over time
- Rest and recover
- Choose different room qualities
- Track healing progress
- Manage character recovery costs

Features:
- Multiple room tiers with different healing rates
- Cost-based healing system
- Automated progress tracking
- Event logging system
- Weekly stay calculations

Room Tiers and Rates:
- Basic:     0 GP/week  - No healing
- Common:    10 GP/week - 1 week stay
- Quality:   100 GP/week - 3 weeks stay
- Luxury:    200 GP/week - 7 weeks stay
- Royal:     500 GP/week - 10 weeks stay

### The Temple of Cant
A sacred location where characters can:
- Resurrect fallen party members
- Heal injured characters
- Manage character status effects
- Track character mortality
- Handle character restoration

Features:
- Character resurrection system
- Status management (DEAD, ASHES, LOST, OK)
- Constitution-based revival chances
- Age progression mechanics
- Permanent character loss handling

### Boltac's Trading Post
A merchant interface where characters can:
- Buy and sell equipment
- Manage character inventory
- Pool and distribute party gold
- View available items and their costs
- Track individual character wealth

Features:
- Dynamic inventory management
- Real-time gold tracking
- Party-wide gold distribution
- Persistent inventory saving

### Edge of Town
The gateway to adventures where players can:
- Initiate combat encounters
- Manage party formation
- Navigate the wilderness
- Return to castle/town
- Access the combat interface

Features:
- Monster encounter system
- Combat management
- Party positioning
- Navigation options
- Safe retreat mechanics

## Combat System

### Combat Module
A comprehensive battle system where characters can:
- Engage in turn-based combat
- Execute various combat actions
- Fight monsters from D&D 5E
- Manage party tactics
- Handle combat outcomes

Features:
- Turn-based combat engine
- Multiple action types
- Monster encounter system
- Party management
- Combat UI interface
